# kubernetes-cluster

## Install master

Clone github repository
```bash
git clone https://github.com/Smolinec/kubernetes-cluster.git
```
Change to run permissions
```bash
chmod +x master-install.sh base-install.sh
```
Run install script
```bash
./master-install.sh
```
